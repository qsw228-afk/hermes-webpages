# 平面 UI 设计基本原则与 2026 潮流趋势

更新时间：2026-04-30 17:28:46 CST

## 核心结论

平面 UI 设计的顺序应该是：**先清楚，再漂亮；先可用，再潮流。**

- 平面设计基本功解决：层级、对齐、分组、对比、重复、留白、字体、色彩。
- UI 设计基本功解决：可点击感、状态反馈、无障碍、任务路径。
- 2026 趋势可以借鉴：AI-first、Bento Grid、动效品牌、3D/空间、手工质感、实验字体、负责任玻璃拟态、新粗野主义、包容性无障碍。

## 10 条基本原则

1. **视觉层级**：先让用户知道最重要的是什么。
2. **对齐**：统一起点、统一边距、统一基线。
3. **亲密性/分组**：相关元素靠近，不相关元素拉开。
4. **对比**：用大小、颜色、明暗、疏密突出重点。
5. **重复/一致性**：相同功能用相同样式，相同层级用相同规格。
6. **留白**：留白是注意力预算，不是浪费空间。
7. **字体与可读性**：标题用于识别，正文用于阅读，标签用于扫描。
8. **色彩系统**：颜色先服务状态和语义，再服务装饰。
9. **可点击感**：按钮要像按钮，输入框要像输入框。
10. **反馈与无障碍**：悬停、按下、加载、成功、错误、禁用、焦点状态都要明确。

## 2026 值得关注的趋势

1. **AI-first / 生成式 UI**：界面根据用户和任务动态生成模块，但要保留用户控制与解释。
2. **Bento Grid 信息卡片**：适合复杂信息模块化展示，但大小必须对应信息权重。
3. **动效驱动品牌**：微交互和状态变化成为品牌气质的一部分。
4. **3D / 空间化界面**：用空间层级帮助理解结构，避免纯炫技。
5. **手工质感 / 反 AI 完美感**：纹理、拼贴、颗粒和不完美感制造人味。
6. **实验字体 / 液态 Typography**：字体作为画面主体，但应主要用于标题和关键视觉。
7. **负责任的玻璃拟态**：透明与模糊仍流行，但必须保证可读性和对比。
8. **新粗野主义 + 实用主义**：高对比、硬边框、直接表达，但不能粗糙。
9. **包容性与无障碍优先**：高对比、键盘导航、可缩放字体、降低认知负担。

## 上线前设计体检

- [ ] 首屏 3 秒内能看出页面目的和主行动。
- [ ] 主要内容沿同一网格/边距对齐。
- [ ] 相关信息已经靠近并分组。
- [ ] 主按钮只有一个，并且视觉权重最高。
- [ ] 颜色有语义，不乱用状态色。
- [ ] 正文可读，字号/行高适合移动端。
- [ ] 所有可点击元素看起来可点击。
- [ ] 悬停、按下、加载、错误、成功、禁用状态齐全。
- [ ] 键盘 Tab 可见焦点。
- [ ] 文字与背景对比足够。
- [ ] 组件样式重复一致。
- [ ] 趋势元素服务内容，而不是抢内容。
